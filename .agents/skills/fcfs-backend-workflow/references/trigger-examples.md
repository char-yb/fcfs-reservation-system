# Trigger Examples

Use these examples to sanity-check skill selection.

## Should Trigger

1. "BookingFacade stock flow is failing; fix it."
2. "Add a new payment method to this reservation service."
3. "Review the Redis lock implementation."
4. "Add tests for idempotent booking."
5. "Refactor the JPA product stock repository."
6. "Update the API error response for payment failures."
7. "Run validation for my backend changes."
8. "Refresh the Codex harness for this repo."

## Should Not Trigger

1. "Summarize an unrelated article."
2. "Create a frontend landing page."
3. "Generate a spreadsheet from this CSV."
4. "Inspect a Sentry issue for another service."
5. "Deploy a Next.js app to Vercel."
6. "Translate a document."
7. "Create a Figma design system."
8. "Explain Kotlin generics without using this repository."
